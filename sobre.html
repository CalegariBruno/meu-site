Sobre a página

Unidades

As unidades como px, % e fr são usadas para definir propriedades de tamanho, como largura, altura, margens e preenchimentos.

Por exemplo, algumas unidades utilizadas, font-size: 20px, width: 200px, padding: 0 15%, grid-template-columns: 1fr 1fr.

Media Query

A regra de mídia é usada para aplicar estilos específicos quando a largura da tela é menor ou igual a 450px.

Por exemplo, @media (max-width: 450px) { ... } é usado para definir estilos específicos para telas menores.

Animações e Transformações

A animação é aplicada diretamente aos elementos do cabeçalho (header). Na tag 'header', é definido um estilo CSS que aplica a animação slideInRight.

É definida usando a regra @keyframes, que especifica os estágios da animação (from e to). Em seguida, a animação é aplicada ao header usando a propriedade animation, onde o nome da animação, a duração e a função de temporização são definidos.

SASS

Variáveis são usadas no arquivo SASS ($primary-color: black) para definir valores que podem ser reutilizados em todo o código CSS.

Um mixin (button-styles) é usado para estilizar os botões comum com as propriedades definidas, permitindo a reutilização desses estilos em vários botões.

Flexbox

Flexbox é utilizado no CSS para organizar os itens do menu de navegação horizontalmente, usando a propriedade display: flex.

Por exemplo, ul { display: flex; justify-content: space-around; align-items: center; } organiza os itens da lista de navegação horizontalmente e os centraliza verticalmente.

Gridlayout

O Gridlayout é utilizado no CSS para organizar as especialidades (WordPress e Java) em uma grade de duas colunas.

Por exemplo, .especialidades { display: grid; grid-template-columns: 1fr 1fr; } organiza as especialidades em uma grade de duas colunas, onde cada especialidade ocupa metade da largura da tela.
